## Adding Images

When adding new images:

1. **File Naming Convention:**
   - Use lowercase with hyphens
   - Be descriptive: `component-name-view.jpg`
   - Examples: `moisture-sensor-installed.jpg`, `relay-board-connections.jpg`

2. **Image Specifications:**
   - Format: JPG or PNG
   - Resolution: 1920x1080 (Full HD) or higher
   - File size: <2MB per image (compress if needed)
   - Quality: Clear, well-lit photos

3. **Photo Tips:**
   - Use good lighting
   - Focus on component/connection
   - Include context when helpful
   - Show labels/annotations if needed
   - Take multiple angles

4. **Documentation:**
   - Update this README with new images
   - Reference images in relevant documentation
   - Add captions describing the image

## Image Tools
